# The Wandsome Few

A multiplayer fantasy RPG inspired by classic MMORPG mechanics.
This project is an experimental recreation of an Old School RuneScape-style game architecture with a custom client server implementation.

The goal of this project is to explore:

- Multiplayer game server architecture
- Tick-based simulation systems
- Client/server networking
- Game state synchronization
- Entity and world management

# Project Goals

This project aims to build a simplified MMORPG architecture including:

- Tick-based game server
- Multiplayer networking
- Persistent player state
- Inventory and item systems
- NPC behaviors
- Combat mechanics
- Map regions and pathfinding
- Questing system

The focus is on **understanding large-scale game architecture** rather than creating a commercial product.
